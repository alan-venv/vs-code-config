# Visual Studio Code
My custom visual studio code settings

## Extensions
* Auto Rename Tag
* Better Comments
* Code Spell Checker
* Error Lens
* Material Icon Theme
* One Dark Pro
* Path Intellisense
* Prettier
* Thunder Client

## Fonts
* sudo apt install fonts-firacode
